Esse é um repositório para o projeto final da disciplina Laboratório de Física 1, ministrada no período 1.3 do IMPATECH.

O repositório é gerenciado por mim, Lucas Fraga Damasceno, e os outros integrantes do grupo são José Armando Duarte, Theo Veiga, e Alan Santos.

O projeto consiste em utilizar um sensor MPU6050 para medir a gravidade e utilizar conceitos de cinemática para prever a posição e uma bolinha em um tubo. A posição da bolinha é representada por LEDs em uma fita endereçável.

O projeto é inteiramente em c++.
